## Allow External Sharing for Specific SharePoint Sites
This PowerShell script helps enable "Anyone" sharing for specific SharePoint sites, covering three major use cases:

1. Allow external sharing for a specific SharePoint site.
2. Enable external sharing for multiple SharePoint sites.
3. Restrict sharing for remaining SharePoint sites.

***Sample log file:***

Once the script is run, a comprehensive log file will be generated, listing the changes applied to the external sharing settings for each site.
